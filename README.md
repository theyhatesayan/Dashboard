# Ayan Dashboard

A modern and responsive dashboard built using HTML, CSS, and JavaScript.
It provides a clean UI to visualize data with cards, charts, and tables.

## About Project

This project is focused on building a real-world dashboard layout using core frontend technologies.
It demonstrates skills like DOM manipulation, dynamic data updates, chart integration, and responsive design.

## Features

* Sidebar navigation with active states
* Dashboard cards (Revenue, Users, Orders, Conversion)
* Dynamic chart using Chart.js
* Update button to refresh chart data
* Search functionality to filter table data
* Responsive layout for different screen sizes
* Dark theme UI
* Smooth animations using AOS

## Tech Stack

* HTML
* CSS
* JavaScript
* Chart.js

## Project Structure

* index.html
* style.css
* query.css
* index.js

## What I Learned

* Building dashboard layouts using CSS Grid
* Working with Chart.js for data visualization

* Creating dynamic UI updates
